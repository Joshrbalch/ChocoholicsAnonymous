Use Case: Manage Provider Information
Context: This will allow the operator to change information about providers in the system
Actors: Operator
Main Success Scenario:
1.	ChocAn Data Center is in interactive mode
2.	Operator changes information about Providers
3.	ChocAn Data Center returns into the interactive mode
Extensions:
2.1. Operator adds a provider
2.2. Operator deletes a provider who has left the network
2.3. Operator updates provider records
